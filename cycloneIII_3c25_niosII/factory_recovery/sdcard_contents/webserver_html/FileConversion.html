Board Update Portal

Altera Embedded Systems Development Kit, Cyclone III Edition


Ready What you need to update the board
  1. SRAM Object File (SOF) containing FPGA Image
  2. Executable Linked File (ELF) containing Software Application
    • ELF not required if the design has no software application

Set Convert design to Flash format
Follow these steps to convert SRAM Object File (.SOF) and Executable Linked File (ELF) to Flash Format i.e. format required to program Flash

Go Program flash
To update the flash memory and reconfigure the FPGA
  1. Use the Browse buttons to browse to the HW and SW Files respectively
  2. Click UPLOAD
  3. Click Reset to configure the FPGA with new image



The contents of this website are contained on a FAT16 formatted SD card in a directory at the top level called "webserver_html". You can replace this content or add new content by placing your SD card into an SD card reader and your content into this directory.

Note: Content you add must be of a known "type" to be served properly. HTML, GIF, and JPEG images are a few examples of types supported by this example application. For more advanced file types you'll need to modify the web-server example application.
