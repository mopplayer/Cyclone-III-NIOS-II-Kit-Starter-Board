Launch Shell Launch Nios II Command Shell
From the Start Menu, select All Programs, Altera, Nios II EDS, Nios II Command Shell   

HW Image Convert hardware image to flash format
In the Nios II Command Shell, navigate to the location of the SOF file and type
sof2flash --activeparallel --timestamp --input="your SOF.sof" --output="your SOF_HW.flash"        --offset=0xe00000
 

SW Image Convert software image to flash format
In the Nios II Command Shell, navigate to the location of the ELF file and type
elf2flash --base=0x04000000 --end=0x4FFFFFF --reset=0x4240000 --input=="your ELF.elf"
--output=="your ELF_SW.flash"
--boot=$SOPC_KIT_NIOS2/components/altera_nios2/boot_loader_cfi.srec





The contents of this website are contained on a FAT16 formatted SD card in a directory at the top level called "webserver_html". You can replace this content or add new content by placing your SD card into an SD card reader and your content into this directory.

Note: Content you add must be of a known "type" to be served properly. HTML, GIF, and JPEG images are a few examples of types supported by this example application. For more advanced file types you'll need to modify the web-server example application.
