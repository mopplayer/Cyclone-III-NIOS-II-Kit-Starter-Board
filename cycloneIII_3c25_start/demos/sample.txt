This is a sample text file provided for the example of writing a file to memory using Altera Cyclone III Starter Kit Control Panel application. If the memory written to is Flash thne the volatile contents will persist upon power cycling the board, However this file can be erased by clicking the Erase button. Other memories will erase themselves once the board is powered down.Memory locations can be editted one at a time by clicking in the individual cells, entering the correct data format Hex values and then pressing enter. Memory contents may be saved by using the Save button then setting the desired boundaries to be saved.
